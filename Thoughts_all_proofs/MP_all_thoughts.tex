\section{Möbius-Ticks: Linear-Time 3SAT in FA}

\begin{theorem}[Möbius-Ticks Correctness in FA]
The following algorithm decides any 3SAT instance with $n$ variables and $m$ clauses in time $O(n m)$:
\begin{lstlisting}[language=Python, caption={Möbius-Ticks 3SAT Solver}]
def mobius_3sat(n, clauses):
    for t in {0, 1}:
        for k in range(n):
            assign = [bool(((k >> i) % 2) ^ (t % 2)) for i in range(n)]
            if all(any(assign[abs(lit)-1] == (lit > 0) for lit in c) for c in clauses):
                return assign
    return None
\end{lstlisting}
\end{theorem}

If $C$ is unsatisfiable, it returns None (intuitionistic statement “no witness exists”).

\[
a_i^{(t,k)} = \left\lfloor k / 2^i \right\rfloor \bmod 2 \oplus (t \bmod 2).

\begin{proof}
